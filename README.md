# Google Play Store Data Analysis

This project explores and analyzes the Google Play Store dataset using Python and Jupyter Notebook.

## Project Overview

The notebook includes:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of app categories, ratings, installs, and reviews
- Insights into trends within Google Play Store applications

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib

## Dataset

- googleplaystore.csv

## Files

- `google-playstore-data-analysis.ipynb`
- `googleplaystore.csv`
